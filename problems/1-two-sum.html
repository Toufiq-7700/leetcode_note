#1 Easy

Two Sum

2025-05-20

Problem Statement

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target. You may assume that each input would have exactly one solution, and you may not use the same element twice.

Input/Output Examples

Example 1
Input:nums = [2,7,11,15], target = 9 Output:[0,1]
Because nums[0] + nums[1] == 9, we return [0, 1].
Hash Table lookup simulation: Hash Map: {value : index} Iteration 0: num = 2, complement = 9 - 2 = 7 - 7 not in hash map -> insert {2 : 0} Iteration 1: num = 7, complement = 9 - 7 = 2 - 2 IS in hash map! Value 2 is found at index 0 - Return [0, 1]

Complexity Analysis

Time
O(n)
Single pass. Each hash table lookup is O(1).
Space
O(n)
Hash table stores at most n elements.

Approach: One-Pass Hash Table

Create a hash map to store {value : index}.

1. Iterate with index i and element num.
2. Calculate complement = target - num.
3. If complement exists in seen, return [seen[complement], i].
4. Otherwise insert seen[num] = i.

Wrong Attempts

Nested loops O(N²) brute force — bad time complexity. Hash map optimizes to linear.

Optimal Implementation

python 
def twoSum(self, nums: List[int], target: int) -> List[int]:
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

Key Takeaway

Search matching pair in unsorted array = think Hash Map
