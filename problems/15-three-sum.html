3Sum

2025-05-22

Problem Statement

Given an integer array nums, return all the triplets [nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0. The solution set must not contain duplicate triplets.

Input/Output Examples

Example 1
Input:nums = [-1,0,1,2,-1,-4] Output:[[-1,-1,2],[-1,0,1]]
The unique triplets are [-1,0,1] and [-1,-1,2].
Sorted: [-4, -1, -1, 0, 1, 2] Fixed Pointer: nums[i] = -1 Scanning sub-array: [-1, 0, 1, 2] ^ ^ left right -> sum = 0 (Match! Add [-1, -1, 2])

Complexity Analysis

Time
O(n²)
Sorting O(n log n) + nested loops O(n²) for fixed element + two pointers.
Space
O(1)
Constant extra memory (not counting output list and sort space).

Approach: Sort + Two Pointers

1. Sort the input array.
2. Loop with index i. Skip duplicates.
3. For each i, run Two Pointers on remaining sub-array to find pairs summing to -nums[i].
4. Skip duplicate left/right values after finding a match.
5. Adjust pointers based on sum comparison.

Wrong Attempts

Brute force triple loops O(N³) times out. Storing triplets in set is messy for dedup.

Optimal Implementation

python 
def threeSum(self, nums: List[int]) -> List[List[int]]:
    res = []
    nums.sort()
    
    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i-1]:
            continue
            
        left = i + 1
        right = len(nums) - 1
        
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            
            if total == 0:
                res.append([nums[i], nums[left], nums[right]])
                while left < right and nums[left] == nums[left + 1]:
                    left += 1
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1
                left += 1
                right -= 1
            elif total < 0:
                left += 1
            else:
                right -= 1
                
    return res
