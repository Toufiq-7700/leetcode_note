Two Sum II - Input Array Is Sorted

2025-05-23

Problem Statement

Given a 1-indexed sorted array of integers, find two different numbers that add up to a target number. Return their indices incremented by 1 as an array of length 2.

Input/Output Examples & Visualizations

Example 1
Input: numbers = [2,7,11,15], target = 9 Output: [1,2]
The sum of 2 and 7 is 9. Therefore index1 = 1, index2 = 2. We return [1, 2].
Simulation: [ 2 , 7 , 11 , 15 ] target = 9 ^ ^ left right -> sum = 17 (too large, right pointer moves left) [ 2 , 7 , 11 , 15 ] target = 9 ^ ^ left right -> sum = 13 (too large, right pointer moves left) [ 2 , 7 , 11 , 15 ] target = 9 ^ ^ left right -> sum = 9 (Match! Return index1 = left+1, index2 = right+1)

Complexity Analysis

Time Complexity
O(n)
Each element is scanned at most once since left and right pointers only move towards each other.
Space Complexity
O(1)
Only two pointer variables are allocated. No extra dynamic space is used.

Approach & Intuition: Two Pointers

Initialize two pointers: `left` at the beginning (0) and `right` at the end (length-1) of the array.

1. Calculate the sum of the elements at these pointers: current_sum = numbers[left] + numbers[right].
2. If current_sum == target, return [left + 1, right + 1] since the problem is 1-indexed.
3. If current_sum < target, we need a larger sum, so we increment left.
4. If current_sum > target, we need a smaller sum, so we decrement right.
5. Repeat until the target is found.

Previous / Wrong Attempts Tried

Nested loops brute force O(N²) scan which resulted in Time Limit Exceeded (TLE) on large arrays. Binary search O(N log N) works but is suboptimal.

Optimal Implementation

python 
def twoSum(self, numbers: List[int], target: int) -> List[int]:
    # Initialize left and right pointers
    left = 0
    right = len(numbers) - 1
    
    while left < right:
        current_sum = numbers[left] + numbers[right]
        
        if current_sum == target:
            # Return 1-indexed array
            return [left + 1, right + 1]
        elif current_sum < target:
            # Need a larger sum, move left rightward
            left += 1
        else:
            # Need a smaller sum, move right leftward
            right -= 1
            
    return []
